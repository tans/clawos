# Agent Market Portal

## Scripts

- `bun install`
- `bun run dev`
- `bun run build`
- `bun test`

## Deployment notes

- Build output: `agent-market/dist`
- The `web` static `/agent-market` page can link here through `AGENT_MARKET_PORTAL_URL`
- Keep `MARKETPLACE_ENABLED=1` in `web` if you want the main site navigation entry to stay visible

# ClawOS App 当前界面功能清单

> 基于当前 `app/webview` 前端代码梳理，仅列出“界面上已经出现”的功能入口与可见操作。

## 0. 全局界面能力（所有页面共用）

- **左侧主导航（5个主入口）**
  - 控制台
  - 通讯设置
  - 模型配置
  - 功能配置
  - 更多设置
- **顶部全局操作**
  - 刷新页面
  - 启动 openclaw（打开 openclaw 控制台）
- **侧边栏底部版本区**
  - 显示品牌名、域名、本地版本
  - 自动检查更新，发现新版本时可一键启动更新
  - 打开 openclaw 控制台
- **任务日志中心（全局浮动入口）**
  - 显示任务汇总（总数/执行中/失败）
  - 右侧抽屉查看任务列表与实时日志
  - 支持未读日志提示

---

## 1. 控制台（Dashboard）

- **状态条**
  - 版本
  - 执行环境（Windows / WSL）
  - 网关状态
- **常用动作**
  - 启动 openclaw
  - 重启 openclaw
  - 升级 openclaw
- **任务日志卡片**
  - 展示当前任务摘要
  - 打开日志中心

---

## 2. 通讯设置（Channels）

- **飞书配置**
  - 开关启用/禁用
  - 填写 `appId`
  - 填写 `secret`
  - 保存参数
- **企业微信配置**
  - 开关启用/禁用
  - 查看企微网关状态
  - 刷新状态
  - 重启企微网关
- **微信模块（openclaw-weixin）**
  - 开关启用/禁用
  - 自动启动微信扫码登录流程
  - 重新获取登录链接
  - 在系统浏览器打开扫码页面
  - 展示登录状态、账号信息、登录链接与操作提示

---

## 3. 模型配置（Agents）

- **模型路由配置（多 Provider）**
  - 预置/读取多个提供方（如 Kimi、MiniMax、FreeGPT）
  - 每个提供方可配置：
    - Base URL
    - API 类型
    - 当前模型
    - API Key
  - 设置“主模型”
  - 设置“备用模型”
  - FreeGPT 单独支持启用开关（作为 fallback）
  - 按提供方单独保存

---

## 4. 功能配置（Skills）

- **MCP 技能中心展示**
  - Windows MCP
  - 影刀 MCP
  - 微信 MCP
  - CRM MCP
- **功能配置项**
  - “开放所有权限”开关（Tools profile）
  - 保存功能配置
- **后台跳转**
  - 一键前往 openclaw 后台配置 Skills
  - 显示跳转状态

---

## 5. 更多设置（聚合页）

该页会顺序展示三个模块：**服务参数**、**环境安装**、**配置备份**。

### 5.1 服务参数（Settings）

- openclaw Token 编辑与保存
- 授权控制地址编辑与保存（含格式校验）
- Company 通讯地址编辑与保存
- 开机自动启动桌面客户端开关
- 重新加载设置

### 5.2 环境安装（Environment）

- **Windows / WSL 双目标环境安装**
  - Python 安装
  - uv 安装
  - bun 安装
  - 展示已安装版本/状态
- **ClawHub 安装**
- **MCP 构建管理**
  - windows-mcp
  - yingdao-mcp
  - wechat-mcp
  - crm-mcp
  - 按项触发构建并显示状态
- 刷新环境状态
- 刷新 MCP 状态
- 任务日志入口（打开日志中心）

### 5.3 配置备份（Backups）

- 读取并展示 openclaw 配置备份列表
- 选择备份文件
- 查看备份时间与大小
- 刷新备份列表
- 执行“切换到该备份”（回滚）
- 任务日志入口（打开日志中心）

---

## 6. 功能配置页内的扩展模块（同属“功能配置”主导航）

> 在“功能配置”主导航下，除了 Skills，还会同时渲染以下模块：浏览器设置、桌面控制、本地钱包。

### 6.1 浏览器设置（Browser）

- 配置并保存 CDP 端口（写入 browser.cdpUrl 的端口位）
- 重新加载浏览器配置
- 浏览器动作：
  - 检测
  - 修复
  - 打开 CDP
- 展示当前 cdpUrl
- 任务状态与日志同步到日志中心

### 6.2 桌面控制（Desktop Control）

- 桌面 MCP 服务开关（启动/停止）
- 展示运行状态、监听地址与接口 URL
- 任务状态跟踪与日志中心联动

### 6.3 本地钱包（Wallet）

- 生成本地钱包
- 刷新钱包状态
- 展示钱包地址、创建时间
- 展示“本次生成私钥”（仅生成后显示）
- 查看多链余额：
  - ETH 主网（ETH / USDT）
  - BSC（BNB / USDT）
  - Base（ETH / USDT）
  - 支持余额获取异常提示
